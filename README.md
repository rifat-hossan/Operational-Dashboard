## 📌 Project Overview

The Operational Dashboard is designed to provide a structured and intuitive interface for managing operational workflows.  

This design focuses on:
- Clear visual hierarchy  
- Consistent design system  
- Scalable component structure  
- User-friendly navigation  
- Data-focused layout  

---

## 🧩 Key Pages Included

- Authentication (Login, Registration, OTP, Reset Password)
- Dashboard Overview
- Analytics
- Rewards Management
- Customer Management
- Subscription
- Settings
- Support

---

## 🛠 Design Principles

- 🎯 Clarity over complexity  
- 📐 Structured spacing & layout system  
- 🎨 Consistent color & typography system  
- 🔄 Reusable components  
- 📊 Data readability & usability
